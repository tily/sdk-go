ニフティクラウド APIリファレンス

クラウド トップ>API>DNS>GetChange

DNS：GetChange

処理概要

更新リクエスト情報を取得します。

情報を取得する更新リクエスト情報は、リクエスト内で更新リクエストIDを指定することで行います。

リクエストURL

https://dns.api.cloud.nifty.com/2012-12-12N2013-12-16/change/[更新リクエストID]

リクエスト方法

GET

呼び出しタイプ

同期

リクエスト

パラメーター

なし

ボディ

なし

レスポンス

XML

要素名 説明 値 備考
GetChangeResponse ルート －  
ChangeInfo 更新リクエスト情報 －  

エラー

コード メッセージ HTTPステータスコード 説明
NoSuchChange No change found 404 URLパス内に記述された更新リクエストIDが存在しない

備考

本APIは更新系API成功時の更新リクエストIDのみ対応しています。
